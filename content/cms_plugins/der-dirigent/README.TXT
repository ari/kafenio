===============================
Java-wysiwyg-editor patch für 
der-dirigent beta release 1
===============================

INSTALLATION
------------
1. entpacken des archivs
2. die datei backend/inc/fnc.type_forms.php ins verzeichnis 
   <dedi-installations-ordner>/backend/inc/ auf dem webserver hochladen
3. das verzeichnis projek01/cms/wysiwyg in jedes projekt-verzeichnis hochladen
   nach dem upload sollten die projektverzeichnisse je ein verzeichnis cms/wysiwyg/
   enthalten. bitte prüfen ob die verzeichnisse cms und wysiwyg ausführen (x) rechte
   gesetzt haben und ob die dateien von einem browser aus lesbar sind.
   (test: http://<domain>/dedi/project01/cms/wysiwyg/applet.jar zeigt dateiinhalt an.)
4. auf dem webserver im verzeichnis backend/external/wysiwyg/ die datei applet.jar 
   löschen

===============================
Java-wysiwyg-editor patch for 
der-dirigent beta release 1
===============================

INSTALLATION
------------
1. unpack the .zip archive
2. copy the file backend/inc/fnc.type_forms.php to the directory <dedi-install-folder>/backend/inc/
   on your webserver.
3. upload the directory projek01/cms/wysiwyg to each project folder on your webserver 
   and make sure that the directories cms and wysiwyg have execute permissions and
   the files are readable by the webserver. (to check call url: 
   http://<domain>/dedi/project01/cms/wysiwyg/applet.jar)
4. delete applet.jar in the <dedi-install-folder>/backend/external/wysiwyg/ folder on your webserver
